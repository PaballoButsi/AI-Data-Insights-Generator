# AI-Data-Insights-Generator
The AI Data Insights Generator is a no-code Generative AI project that analyzes structured datasets and extracts meaningful business insights. It identifies patterns, evaluates performance, and generates actionable recommendations to support data-driven decision-making.  
🎯 Objective

The goal of this project is to demonstrate how AI can be applied in data analytics by:

Analyzing structured datasets
Identifying trends and patterns
Evaluating product and customer performance
Detecting issues and anomalies
Generating business recommendations
Translating data into simple, actionable insights
Google Sheets Dataset → Google AI Studio → AI Data Analysis → Insight Report
🧠 Key Features
Automated dataset analysis
Trend and pattern detection
Customer feedback interpretation
Product performance evaluation
Business recommendations generation
Simplified summary for non-technical users
🛠️ Tools Used
Google Sheets – Dataset storage and organization
Google Forms – Optional data input collection
Google AI Studio – AI-powered analysis and insight generation
📊 Datasets Used
Dataset 1: Small Dataset (Baseline Analysis)
10 rows of customer product data
Used for basic insight extraction
Dataset 2: Mixed Feedback Dataset
Combination of positive and negative reviews
Used to identify performance issues and customer dissatisfaction
Dataset 3: High-Performance Dataset
Mostly positive ratings
Used to identify success patterns and best-performing products
📈 Example Output
Key Insights Generated:
Products with high ratings consistently have strong customer satisfaction
Negative feedback is often linked to performance and durability issues
Price does not always correlate with customer satisfaction
Smartwatches and laptops show the most variation in user experience
Business Recommendations:
Improve quality control for low-rated products
Focus on battery and performance improvements
Highlight high-performing products in marketing campaigns
Collect more structured customer feedback
🧪 Test Cases
